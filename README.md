# 🛡️ Pro-Python-Sniffer

A professional-grade, lightweight network packet sniffer built with Python and Scapy. Designed for network analysis, security auditing, and educational purposes.

## ✨ Features
- **Real-time Capture:** Monitor network traffic as it happens.
- **Protocol Intelligence:** Automatically detects TCP, UDP, and ICMP protocols.
- **Advanced Filtering:** Support for BPF (Berkeley Packet Filter) syntax.
- **Pcap Export:** Save captured traffic to `.pcap` files for deep analysis in Wireshark.
- **Clean Interface:** Minimalist and readable terminal output.
